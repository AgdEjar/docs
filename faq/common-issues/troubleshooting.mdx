## Common HTTP Errors

### 400 Bad Request
**Problem**: Invalid request parameters or body.

**Solution**:
1. Check request body format
2. Verify all required fields are present
3. Validate data types
4. Check for proper date formats (YYYY-MM-DD)

### 404 Not Found
**Problem**: Resource doesn't exist.

**Solution**:
1. Verify IDs are correct
2. Check if resource was deleted
3. Ensure proper endpoint URL

### 500 Server Error
**Problem**: Server-side issue.

**Solution**:
1. Retry request after a delay
2. Check [Status Page](https://stats.uptimerobot.com/koyjLJpRaj)
3. Contact support if persistent
