## Base URL

All API requests should be made to:

```bash
https://app.agdejar.sa
```

<Note>
  Make sure to use HTTPS for all API calls to ensure secure communication.
</Note>

## Authentication

Our API uses OAuth 2.0 Bearer token authentication. All API calls require an authentication token.

### Available Endpoints
- [Get OAuth Token](/api-reference/auth/oauth-token) - Obtain access token
- [Get Client Info](/api-reference/auth/client) - Verify client setup

## Property Management

Comprehensive property management including deed verification and property details.

### Available Endpoints
- [Search Deed](/api-reference/properties/search-deed) - Search for property deed
- [Sync Deed](/api-reference/properties/sync-deed) - Synchronize deed information
- [Search Deed Remote](/api-reference/properties/search-deed-remote) - Remote deed search
- [List Properties](/api-reference/properties/list-properties) - Get all properties
- [Get Property](/api-reference/properties/get-property) - Get property details
- [Update Property](/api-reference/properties/update-property) - Update property information
- [Delete Property](/api-reference/properties/delete-property) - Remove property

## Unit Management

Manage individual units within properties.

### Available Endpoints
- [List Units](/api-reference/units/list-units) - Get all units
- [Get Unit](/api-reference/units/get-unit) - Get unit details
- [Update Unit](/api-reference/units/update-unit) - Update unit information
- [Delete Unit](/api-reference/units/delete-unit) - Remove unit

## Location Management

Manage cities and regions for property addresses.

### Available Endpoints
- [List Cities](/api-reference/cities/list-cities) - Get all cities
- [Get City](/api-reference/cities/get-city) - Get city details
- [List Regions](/api-reference/regions/list-regions) - Get all regions
- [Get Region](/api-reference/regions/get-region) - Get region details

## Party Management

Manage property owners, tenants, and other parties.

### Available Endpoints
- [Search Party](/api-reference/parties/search-party) - Search for parties
- [Update Party](/api-reference/parties/update-party) - Update party information
- [Delete Party](/api-reference/parties/delete-party) - Remove party

## Contract Management

Complete contract lifecycle management.

### Available Endpoints
- [Download Contract PDF](/api-reference/contracts/download-pdf) - Get contract document
- [List Contracts](/api-reference/contracts/list-contracts) - Get all contracts
- [Create Contract](/api-reference/contracts/create-contract) - Create new contract
- [Update Contract](/api-reference/contracts/update-contract) - Update contract details
- [Delete Contract](/api-reference/contracts/delete-contract) - Remove contract

## Response Format

All responses are returned in JSON format. Successful responses will have a 200 status code.

## Rate Limiting

Please contact support for information about rate limits for your specific use case.

## Error Handling

The API uses standard HTTP response codes:
- 200: Success
- 400: Bad Request
- 401: Unauthorized
- 403: Forbidden
- 404: Not Found
- 500: Internal Server Error
